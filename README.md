#Bus Ticket Reservation System 

A web-based application developed using Spring Boot and MySQL to manage bus details, ticket price and tracking seat availability.

## Technologies Used 
- Java
- Spring Boot
- Thymeleaf
- MySQL
- HTML, CSS

## Features
- Add, Update, delete nus details
- Search buses by source and destination
- Seat availability management
## How to Run
1. Import the project into Spring Tool
2. Configure MySQL database
3. Run the Application
4. Access via localhost in browser
